Hooray!

When comments count is turned on you can search using special keywords.

  • Use : to show comments with timestamps (0:37)
  • Use :all to show all comments.
  • Use :link to show comments with links.
  • Use :reply to show comments with replies.
  • Use :creator to show comments by the uploader.
  • Use /.../ for RegExp search.

Global Search - Available on all videos:

  • Type global: followed by keywords to search all channel videos.

General Settings:

  • Type /commands to change keyboard shortcuts.

Keyboard Shortcuts

Few things to take in mind before setting a shortcut:

  • Shortcut used by other app needs to be released first.
  • You can use the Shift + X pattern.

Youtube API Key

Youtube API key is needed for any software that wants to access youtube data like comments and videos.

Some apps use their own key for this purpose, but that may have disadvantages for the end-user in terms of privacy and reliability.

Generating your own key is free and ensures your activity stays private and not limited.
